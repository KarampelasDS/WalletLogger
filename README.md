# Wallet Logger (Alpha MVP)

**Wallet Logger** is a mobile-first expense manager for Android devices.

> ⚠️ Alpha MVP release — core functionality works, but many advanced features are not included yet.

---

## Features in this Alpha
- Multi-currency support
- Accounts with balances
- Income & expense categories
- Step-by-step setup wizard
- Offline-ready via local SQLite storage

❌ Not included in this alpha:
- Data analytics or reporting
- Import/export of data

---

## Installing the APK
1. Download the APK from the [Releases page](https://github.com/KarampelasDS/WalletLogger/releases).  
2. On your Android device, allow installation from unknown sources if prompted.  
3. Open the APK to install Wallet Logger.  

> Only tested on Android. No iOS build included.
